# Example Article

This is an example article written in Markdown. You can use all standard Markdown syntax here.

## Headings

Use headings to structure your content. This is a level 2 heading.

### Level 3 Heading

And so on.

## Paragraphs

Write paragraphs normally. They will be automatically formatted with proper spacing and line height.

## Lists

You can create unordered lists:
- First item
- Second item
- Third item

Or ordered lists:
1. First step
2. Second step
3. Third step

## Code

You can include `inline code` with backticks.

Or code blocks:

```python
def hello_world():
    print("Hello, World!")
```

## Links

You can add [links](https://example.com) inline.

## Images

Images should be placed in `assets/images/` and referenced like:

![Example Image](./example-image.png)

## Blockquotes

> This is a blockquote. Use it for quotes or emphasis.

## Emphasis

You can use **bold** and *italic* text for emphasis.
